# Retired Sprint Queue

Last updated: 2026-04-27

The old sprint-row protocol is retired. New durable work is tracked as execution plans:

- Active plans: `docs/exec-plans/active/`
- Completed plans: `docs/exec-plans/completed/`
- Plan templates: `docs/exec-plans/templates/`

Historical sprint evidence remains available through git history, `docs/SPRINT_ARCHIVE.md`, `docs/WORKLOG.md`, and `docs/CANONICAL_REPORT.md`.
